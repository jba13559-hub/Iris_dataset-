# Iris Flower Classification

## Project Overview
This project uses machine learning to classify iris flowers into three species: Setosa, Versicolor, and Virginica based on sepal and petal measurements.

## Dataset
The Iris dataset contains 150 samples with the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Data Preprocessing
- Checked for missing values
- Performed train-test split
- Applied feature scaling when required

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier

## Evaluation Metrics
- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Results
Among the tested models, Logistic Regression achieved the best performance and accurately classified most flower species.
